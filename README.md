# basenext

A NextJS boilerplate for building and shipping AI SaaS applications quickly.

[basenext](https://github.com/toolsdk-ai/basenext) provides a comprehensive set of templates and components to accelerate AI app development.

**Key Features:**
- TypeScript-first architecture
- Built for NextJS with turbopack/webpack support
- Ready-to-use AI SaaS components
- Optimized for rapid development and deployment
